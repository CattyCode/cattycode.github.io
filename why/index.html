Why CattyCode?

CattyCode is a modern, lightweight block editor built for real coding logic – not games or sprites.

🟢 CattyCode vs 🔵 Scratch

Feature / Philosophy 🔵 Scratch 🟢 CattyCode
Game/Animation Focus ✅ ❌ (by design)
Beginner-friendly interface ✅ ✅
Real-world code structure ❌ ✅
Easy to understand flow ❌ ✅
Lightweight and fast ❌ ✅ (tiny editor)
Sprite-based distractions ✅ ❌
Customizable (Blockly-based) ❌ ✅
Made for teaching programming ⚠️ (more creative-focused) ✅

🚀 What Makes CattyCode Different?

  • 🧩 Built with Blockly, by Google
  • 💡 No sprites or stages – just pure logic
  • 👶 Friendly, but grows with you into real coding
  • 💻 Tiny HTML app – works great on low-end devices (like Little10 😄)

✅ Great For...

  • Practicing loops, conditionals, functions, and variables
  • Learning core logic without distractions
  • Teaching programming before diving into real languages

🔒 Safe & Clean

  • No raw JS blocks (like `eval`)
  • No sprite/costume spam
  • No confusing visual clutter

💡 CattyCode is made to help you think like a programmer — not just play like one.
